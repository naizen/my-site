Alfred

Alfred is a real-time chatbot assistant that incorporates natural language processing to handle tasks. It uses the wit.ai engine to parse text and respond with the appropriate response.

Alfred is implemented using React on the frontend, Pusher Chatkit, and wit.ai.

Website Repository

FoodMatcher

FoodMatcher is a Tinder-inspired app that helps you figure out where you want to eat by matching you with food places nearby.

FoodMatcher is implemented using React Spring, React-use-gesture, Redux for state management, geolocation and the Yelp API to find food places near your location.

Website Repository

Movie Library

MovieLibrary is a web application where you can explore movies in theaters, popular, and upcoming. You can also search for movies, watch trailers, and view detailed movie info.

It was built with React.js on the frontend and Apollo GraphQL. It uses TheMovieDB (TMDB) API for movie data.

Website Repository

Taskboard

Taskboard is a simple Trello-inspired productivity application that allows users to add, remove, search, label, and reorganize lists and cards.

It was built using a Node Express.js server, MongoDB for the backend, React and Redux for frontend state management.
